## Setup project

1. Clone project  
  `$ get clone https://github.com/nakulkundaliya/react-native-demo-task`

2. Instal node dependency  `$ npm i`

3. Run Project `$ react-native run-android`

4. Run Project in ios `$ cd ios pod install ` then
  `$ react-native run-ios`
